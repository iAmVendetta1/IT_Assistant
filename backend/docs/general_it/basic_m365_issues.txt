UPDATED: 2026-02-18

============================================================
SECTION 1 — ACCOUNT & LOGIN ISSUES
============================================================

ISSUE: User cannot sign in to Microsoft 365.
CAUSES:
- Incorrect password.
- Account locked due to failed attempts.
- MFA required but not completed.
- License missing or expired.

RESOLUTION:
1. Verify username and password.
2. Check Azure AD sign-in logs for failures.
3. Reset password if needed.
4. Confirm MFA device is available and configured.
5. Verify user has an active M365 license assigned.
6. Check Conditional Access policies blocking login.

------------------------------------------------------------

ISSUE: MFA not prompting or user cannot complete MFA.
CAUSES:
- Authenticator app not registered.
- Phone number outdated.
- Push notifications disabled.

RESOLUTION:
1. Reset MFA via Azure AD → User → Authentication Methods.
2. Re-register Authenticator app.
3. Add backup phone number.
4. Ensure device has internet and notifications enabled.

============================================================
SECTION 2 — OUTLOOK & EMAIL ISSUES
============================================================

ISSUE: Outlook stuck on “Trying to connect”.
CAUSES:
- Network connectivity issues.
- Autodiscover failure.
- Corrupt Outlook profile.

RESOLUTION:
1. Test network access to outlook.office365.com.
2. Run Test Email AutoConfiguration.
3. Create a new Outlook profile.
4. Check for firewall or proxy blocking M365 endpoints.

------------------------------------------------------------

ISSUE: User not receiving emails.
CAUSES:
- Inbox rules moving mail.
- Junk filtering.
- Blocked senders.
- Mailbox full.
- Transport rules.

RESOLUTION:
1. Check Outlook rules.
2. Review Junk Email folder.
3. Check Blocked Senders list.
4. Verify mailbox quota.
5. Review Exchange Admin Center message trace.

------------------------------------------------------------

ISSUE: Shared mailbox not showing in Outlook.
CAUSES:
- Missing permissions.
- Auto-mapping disabled.
- Outlook profile corruption.

RESOLUTION:
1. Assign Full Access and Send As permissions.
2. Remove and re-add permissions to trigger auto-mapping.
3. Manually add shared mailbox in Outlook.
4. Recreate Outlook profile if needed.

============================================================
SECTION 3 — TEAMS ISSUES
============================================================

ISSUE: Teams not signing in.
CAUSES:
- Cached credentials.
- Network restrictions.
- Conditional Access.

RESOLUTION:
1. Sign out and quit Teams.
2. Clear Teams cache folder.
3. Verify M365 login works in browser.
4. Check CA policies for Teams-specific blocks.

------------------------------------------------------------

ISSUE: Teams meetings not appearing in Outlook.
CAUSES:
- Teams add-in disabled.
- Wrong Outlook profile.
- Calendar permissions.

RESOLUTION:
1. Enable Teams add-in in Outlook COM Add-ins.
2. Restart Teams and Outlook.
3. Ensure user is signed into the same account in both apps.

============================================================
SECTION 4 — ONEDRIVE & SHAREPOINT ISSUES
============================================================

ISSUE: OneDrive not syncing.
CAUSES:
- Outdated client.
- Path length issues.
- Invalid characters.
- Storage quota exceeded.

RESOLUTION:
1. Update OneDrive client.
2. Shorten file paths.
3. Remove invalid characters.
4. Check OneDrive storage usage.
5. Reset OneDrive client if needed.

------------------------------------------------------------

ISSUE: SharePoint access denied.
CAUSES:
- Missing permissions.
- External sharing disabled.
- Conditional Access.

RESOLUTION:
1. Verify user permissions on the site.
2. Check sharing settings at site and tenant level.
3. Review CA policies affecting SharePoint.

============================================================
SECTION 5 — LICENSING ISSUES
============================================================

ISSUE: User cannot access apps (Word, Excel, Outlook).
CAUSES:
- License not assigned.
- License not activated.
- Sign-in mismatch.

RESOLUTION:
1. Assign appropriate M365 license.
2. Sign out and sign back into Office apps.
3. Run Office activation troubleshooter.
4. Verify user is not using a personal Microsoft account.

============================================================
SECTION 6 — DEVICE MANAGEMENT (INTUNE)
============================================================

ISSUE: Device not showing in Intune.
CAUSES:
- Not enrolled.
- Wrong user signed in.
- Enrollment restrictions.

RESOLUTION:
1. Check Azure AD device list.
2. Confirm user is allowed to enroll devices.
3. Re-enroll via Company Portal.

------------------------------------------------------------

ISSUE: Policies not applying.
CAUSES:
- Device not compliant.
- Policy conflicts.
- Sync delay.

RESOLUTION:
1. Force sync from Company Portal.
2. Check compliance policies.
3. Review Intune troubleshooting logs.

============================================================
SECTION 7 — COMMON ADMIN TASKS
============================================================

TASK: Reset user password.
STEPS:
1. Azure AD → Users → Select user.
2. Click Reset Password.
3. Provide temporary password to user.

------------------------------------------------------------

TASK: Assign or remove licenses.
STEPS:
1. Microsoft 365 Admin Center → Users → Active Users.
2. Select user.
3. Edit license assignments.

------------------------------------------------------------

TASK: Run message trace.
STEPS:
1. Exchange Admin Center → Mail Flow → Message Trace.
2. Enter sender/recipient.
3. Review delivery status.
