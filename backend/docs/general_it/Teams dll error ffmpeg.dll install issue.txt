Issue: 

Downloaded teams installer (MSTeamsSetup) from Microsoft website. 
Attempted to run the setup. 
Received error: "The code execution cannot proceed because ffmpeg.dll was not found."

Resolution:

Uninstall all teams related programs, reboot, and retry MSTeamsSetup. 

If this doesn't work, install Teams directly from the Microsoft Store app.

If Microsoft Store app does not show up or is missing,  open "Add or Remove Programs" click the dots next to Microsoft Store, click advanced, then scroll down and click reset. 
Then you should be able to open Microsoft Store and successfully install Teams.
